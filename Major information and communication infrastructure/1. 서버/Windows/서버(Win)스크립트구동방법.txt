1. 진단 서버에서 [Administrator] 관리자 계정으로 로그인

2. 루트 디렉토리(최상위 디렉토리)에 think 디렉터리 생성
   -> C:\think

3. 백신프로그램 일시 정지
   -> 진단 스크립트 업로드 또는 압축 해지 후 백신 프로그램이 진단 스크립트 파일을 바이러스 프로그램으로 인식하여 자동 삭제할 수 있습니다. 진단 스크립트 실행 시간은 1~3분 정도 소요 되므로 백신 프로그램을 잠시 종료 한 후 작업을 진행해 주시기 바랍니다. 

4. 서버의 C:\think폴더에 스크립트 파일 업로드
   -> C:\think 폴더 안에 "script"폴더와 "스크립트명.bat" 파일을 업로드함.

5. 스크립트를 실행
   -> "C:\think\스크립트명.bat" 파일을 실행
	(C:\think 폴더 내에는 "script"폴더와 "스크립트명.bat"이 모두 같이 있어야 함)

5. 진단 후 결과파일 다운로드 및 전달
   -> 결과파일 예) hostname-Version.txt

### 모든 작업 완료 후 진단 스크립트를 삭제하여 주시기 바랍니다.
